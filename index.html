开源的都是一些Demo类，工具类，自己实验性的项目，后续会开源更多的项目。

  1. iBeaconDemo

    GitHub

    一个beaconAPI相关的demo，如果用beacon做室内定位的话可能会用到。相关类:CLBeacon，CLBeaconRegion，CLLocationManager。

  2. CategoryUtility

    GitHub

    用于在Category添加属性。由于Category不能添加变量，所以要添加属性比较困难。这个工具用了关联对象的办法实现添加属性，并且支持weak。

  3. TestResponder

    GitHub

    用于测试touch透传的Demo，可以实现类似地图的需求。一个根vc不动，上面可以覆盖各种navigation，并且可以将touch事件透传到根vc上。
